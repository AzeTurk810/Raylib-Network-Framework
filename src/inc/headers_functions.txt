# RNF Framework Structure

---

# /Core

The Core folder contains the main networking system of the framework. Every other module depends on it.

## Library 1 : Server.h (Dedicated Server Management)

**Goal:**
Handles everything related to hosting and managing a multiplayer server.

**Functions**

* Create()
* Stop()
* Poll()
* Broadcast()
* Send()
* Kick()
* DisconnectClient()
* DisconnectAll()
* PlayerCount()
* Clients()
* Running()
* GetPort()
* SetMaxPlayers()

---

## Library 2 : Client.h (Client Networking)

**Goal:**
Provides an easy interface for connecting to servers and communicating with them.

**Functions**

* Connect()
* Disconnect()
* Reconnect()
* Poll()
* Send()
* Connected()
* GetPing()
* GetServerAddress()
* ConnectionState()

---

## Library 3 : Events.h (Event System)

**Goal:**
Converts low-level ENet events into easy-to-use callbacks.

**Functions**

* OnConnect()
* OnDisconnect()
* OnPacket()
* OnTimeout()
* OnReconnect()
* OnError()
* OnPing()
* RemoveListener()

---

## Library 4 : Packet.h (Packet Management)

**Goal:**
Creates, sends, receives, and manages network packets.

**Functions**

* SendPacket()
* BroadcastPacket()
* RegisterPacket()
* GetPacketID()
* CreatePacket()
* DestroyPacket()

---

## Library 5 : Serialization.h (Serialization System)

**Goal:**
Converts C++ objects into byte arrays and restores them back.

**Functions**

* Serialize()
* Deserialize()
* Write()
* Read()
* Compress()
* Decompress()
* ValidatePacket()

---

## Library 6 : Memory.h (Memory Management)

**Goal:**
Optimizes memory allocation for networking.

**Functions**

* AllocatePacket()
* FreePacket()
* BufferPool()
* ObjectPool()
* ClearPools()

---

## Library 7 : Threading.h (Thread Management)

**Goal:**
Runs networking independently from the game loop.

**Functions**

* StartNetworkThread()
* StopNetworkThread()
* Lock()
* Unlock()
* PushTask()
* Wait()

---
